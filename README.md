# 🎓 School Management System (PHP + MySQL)

A web-based School Management System developed using **PHP and MySQL**.  
This project manages different departments like **Library, Laboratory, Subjects**, and includes **user authentication (login/signup)** and admin functionalities.

---

## 🚀 Features

### 🔐 Authentication System
- User Signup (`signup.php`)
- User Login (`login.php`)

### 🏫 Modules

#### 📚 Library Management
- Add and manage books
- View records (`library_records.php`)
- Database connection (`library connect.php`)

#### 🧪 Laboratory Management
- Laboratory records (`labrotary.php`)

#### 📖 Subjects Section
- Physics (`physics.php`)
- Chemistry (`chemistry.php`)
- Computer (`computer.php`)

#### 🛠️ Administration
- Admin panel (`administration.php`)
- Manage system data

---

## 🔄 CRUD Operations

- ➕ Create records  
- 📋 Display records (`show_records.php`)  
- ✏️ Update records (`update.php`)  
- ❌ Delete records (`delete.php`)  

---

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS  
- **Backend:** PHP

## ⚙️ Setup Instructions

1. **Clone the repository**

git clone https://github.com/your-username/college-management-system.git

2. **Move project to server directory**

XAMPP → htdocs

WAMP → www

3. **Start Apache & MySQL**

4. **Setup Database**

4. **Run**:

create_db.php

db and table(library).php

Or manually create database in phpMyAdmin

5. **Run the project**

http://localhost/your-folder-name/index.php
